## Introduction

Dojo Blog is a basic blog built with React. You can add/delete entries to the forum. There is a create, home, blog with id and 404 catch page.

## Technologies 
  
 - Javascript
 - React
 - Html 
 - CSS
 - bootstrap



 
 ### Setup 
 
 To run this project, install it locally using npm:

```
$ cd ../lorem
$ npm install
$ npm start
```
